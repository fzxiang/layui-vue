| disabled                         | 禁用选择                                  |  --            | --            | --            |
| placeholder                      | 输入提示                                  |  --            | --            | --            |
| checkStrictly                    | 禁用级联复选                              |  --            | --            | --            |
| collapseTagsTooltip              | 折叠提示                                  |  --            | --            | --            |
| checkStrictly                    | 禁用级联复选                              |  --            | --            | --            |
| minCollapsedNum                  | 超过指定标签后开启折叠                     |  --            | --            | --            |
| size                             | 尺寸大小                                  |  --            | --            | --            |
